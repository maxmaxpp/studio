# **App Name**: Heymaxx

## Core Features:

- Niche Display: Showcase the different niches (Data entry, social media manager, graphic design, development) with descriptions and project examples.
- Project Gallery: Display images and descriptions of projects and client testimonials for each niche.
- Testimonials: Display client testimonials.
- Contact Form: A contact form where potential clients can directly reach out.
- About Section: An about section to provide background information and establish credibility.

## Style Guidelines:

- Primary color: Light tan (#ECE0D1) to provide a soft, neutral background, creating a relaxed viewing experience.
- Background color: Off-white (#ffffff) providing a crisp contrast against other elements.
- Accent color: Deep brown (#38220f) to highlight important details, while staying harmonious with the tan color.
- Body and headline font: 'PT Sans' for a clean, readable and modern aesthetic.
- Minimalist icons to represent each niche, reinforcing clarity.
- Clean and modern layout emphasizing visual content, ensuring a positive user experience.
- Subtle transitions for image galleries.
